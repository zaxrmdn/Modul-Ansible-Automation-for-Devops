#Installing Ansible
---
---
##Installing on Debian/Ubuntu
```
$ apt-add-repository -y ppa:ansible/ansible
$ apt update
$ apt install -y ansible
```
---
---
## Installing on RHEL/Centos
```
$ yum install -y epel-release
$ yum install -y ansible
```
---
---
## Installing on pip
```
pip install ansible
```
---
---
## Check version ansible
```
$ ansible --version
```
